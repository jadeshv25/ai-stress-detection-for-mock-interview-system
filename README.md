# AI Mock Interview System

## Run locally

1. Start the backend in `backend/`.
2. Start the frontend in `frontend/`.
3. If the backend is not running on `http://localhost:8000`, set `VITE_BACKEND_URL` in the frontend environment.

## Notes

- Interview questions and answer analysis now come from the backend API.
- Login and registration now call backend auth endpoints first, with local fallback for offline development.
